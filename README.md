# Triple Espresso

Nome do projeto : Web Project Coffeeshop

Descrição do projeto : A página foi criada utilizando HTML5 para sua estrutura e CSS3 para estilização e organização visual. Funcionalidades : links de navegação funcionais no cabeçalho da página, incorporação de vídeos com a tag iframe, formulário para preenchimento de dados para reserva de mesa e estrutura organizada em seções.

Planos de melhoria do projeto : Adicionar efeitos visuais mais complexos com CSS e tornar o formulário funcional.
